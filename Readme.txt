¡Hola! Bienvenido al repositorio de mi programa que automáticamente une una imagen con un audio, ademas de recortar los silencios en el audio. Este programa se basa en Python y utiliza dos archivos principales: "editingBot.py" y "setup.py".

Para utilizar el programa, primero debes instalar los requisitos necesarios ejecutando el archivo "setup.py". Este archivo instalará todas las bibliotecas requeridas para que el programa funcione correctamente.

- python setup.py

Una vez que hayas instalado los requisitos, puedes usar el archivo "editingBot.py" para unir una imagen con un archivo de audio. El archivo "editingBot.py" toma dos argumentos: el primero es la ruta del archivo de audio que deseas utilizar y el segundo es la ruta de la imagen que deseas utilizar. El programa creará un nuevo archivo que combina la imagen y el audio, y lo guardará en el mismo directorio en el que se encuentra la imagen.

Para usar el programa, simplemente ejecuta el archivo "editingBot.py" desde la línea de comandos y proporciónale los argumentos necesarios. Por ejemplo:

- python editingBot.py path/to/audio/file.mp3 path/to/image/file.png

Ten en cuenta que debes asegurarte de que los archivos de audio e imagen estén en formatos compatibles con el programa.
